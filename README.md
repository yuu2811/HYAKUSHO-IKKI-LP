## ローカル開発

このプロジェクトは静的HTMLサイトのため、シンプルなHTTPサーバーで起動できます。

### 方法1: Python（推奨）
```bash
python -m http.server 8000
```

### 方法2: Node.js (http-server)
```bash
npx http-server -p 8000
```

ブラウザで `http://localhost:8000` にアクセスしてください。
